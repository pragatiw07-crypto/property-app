# Property Management App

A full-stack property management application built using the MERN stack.

## Features

- User Authentication (Login/Register)
- Add Property
- View My Properties
- Delete Property
- Protected Routes using JWT

## Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JSON Web Tokens (JWT)

## Project Structure

property_app/
│── backend/
│── frontend/

## How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
